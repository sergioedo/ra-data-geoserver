# ra-data-geoserver

react-admin data provider for GeoServer. With this datasource, you can connect your GeoServer Features to React-Admin, to view & edit geospatial Data (through WFS service).

# Features

-   [x] List Features
-   [x] View Features
-   [ ] Edit Features
-   [ ] Delete Features
-   [ ] Sorting columns
-   [ ] Complete CRUD for lines & polygons
